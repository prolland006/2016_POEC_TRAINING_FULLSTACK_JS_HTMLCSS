Les principaux standards

  • HTTP est le protocole utilisé pour accéder aux documents du WEB.
  • HTML permet de définir la structure du document.
  • CSS permet de gérer la présentation / mise en forme un document (fonte, marges, positionnement, couleurs, ...) il faut utiliser la technologie CSS.
  • JavaScript permet de rendre dynamique une page web (interactions, évènements, ...).
