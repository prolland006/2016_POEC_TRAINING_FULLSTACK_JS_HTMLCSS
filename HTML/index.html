HTML

HTML est un langage structuré basé sur des balises.

HTML (5) est défini par le W3C et le WHATWG.

Ce qu'est HTML (WhatWG) est défini ici HTML: The Living Standard.

caniuse.com est un bon site pour savoir quel navigateur supporte quelle fonctionnalité...

Généralités sur HTML

Dans la balise <body> se trouve le contenu qui sera visible dans le navigateur.

Penser impérativement à valider vos documents HTML dans votre IDE et par exemple via un validateur HTML.

Les entités HTML permettent d'éviter l'interprétation de certains caractères comme >.

Les commentaires sont délimités par <!-- -->

Utiliser les Content Models pour savoir ce que peuvent contenir les éléments HTML.

form

Les formulaires permettent de saisir des entrées utilisateur.

Attributs intéressants

method
Généralement post ou get
name
Nom du paramètre envoyé au serveur. Requis pour que le champs soit transmis
id
Permet d'associer d'identifier de manière unique des champs de formulaire, pratique pour association avec labels (for) ou manipulation en CSS ou JavaScript
value
La valeur du champs du formulaire. Pratique pour initialiser un champs à une valeur.
placeholder
Indication si champs vide. s'efface à la saisie
