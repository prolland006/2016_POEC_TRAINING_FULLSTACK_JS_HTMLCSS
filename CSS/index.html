Référencer du CSS

Les styles peuvent être définis comme suit, dans l'ordre de surcharge:

  1. La feuille de style 'par défaut' du navigateur
  2. Feuilles de style externes, balise link dans la balise head
  3. Définir des styles localement via la balise style dans la balise head
  4. Inline dans l'attribut style d'un élément HTML

Cascade

Tous les styles "cascadent" du haut vers le bas de la feuille de style.

Ca permet d'ajouter ou de surcharger progressivement des styles.

Box Model

Voir MDN.

Chaque élément a une largeur et une hauteur par défaut.

Les éléments Block ont une largeur de 100%. Ils consomment toute place horizontale disponible.

Les éléments Inline et Inline-block s'étendent et se contractent verticalement par rapport à leur contenu.

Des éléments inline ne vont pas accepter de changement de valeur width et la height.

Des éléments block et inline-block vont accepter des changements de width et de height.

Il est possible de changer la propriété display pour passer d'un block a un inline et vice-versa.

Les éléments peuvent se voir ajuster leur margin (espace entre éléments) et padding (espace entre contenu et bordure)

La propriété box-sizing permet de définir comment les calculs liés au box model (taille d'une élément) sont effectués par les navigateurs.
