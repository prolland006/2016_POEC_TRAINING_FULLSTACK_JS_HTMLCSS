Standard

Voir également sur Wikipedia: HTTP, HTTP 2

Verbes HTTP

GET, POST, PUT, DELETE, TRACE, OPTIONS, HEAD,...

GET Request sont émises lorsque vous rentrez une adresse dans la barre d'adresse ou au click sur un hyperlien.

POST Request sont émises à la soumission d'un formulaire.
